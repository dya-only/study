## 1.1.1 구성요소로 본 인터넷

인터넷은 전 세계적으로 수십억 개의 컴퓨팅 장치를 연결하는 컴퓨터 네트워크이다. \
기존의 이러한 컴퓨팅 장치는 주로 전통적인 데스크톱 PC, 리눅스 워크스테이션, 서버들이었다. \
점점 스마트폰, 태블릿 같은 **이동 인터넷**과 가전제품 같은 비전통적인 인터넷 **사물들**이 모두 인터넷에 연결되고 있다.

> 이러한 모든 장치를 인터넷 용어로 **호스트(host)** 혹은 **종단 시스템(end system)** 이라고 한다.

종단 시스템은 <b>통신 링크(communication link)</b>와 <b>패킷 스위치(packet switch)</b>의 네트워크로 연결된다.
- 이러한 링크는 동축 케이블, 구리선, 광케이블, 라이도 스펙트럼을 포함한 다양한 물리 매체로 구성된다.
- 각 링크들은 다양한 <b>전송률(transmission rate, 링크 대역폭)</b>을 이용하여 데이터를 전송
- 전송률은 초당 비트 수를 의미하는 **bps(bit per second)** 단위를 사용

<br>
<br>

### 패킷 (packet)
한 종단 시스템이 다른 종단 시스템으로 보낼 데이터를 갖고 있을 때,
- 송신 종단 시스템은 그 데이터를 <b>세그먼트(segment)</b>로 나누고 각 세그먼트에 <b>헤더(header)</b>를 붙인다.
   > 이렇게 만들어진 정보 패키지를 <b>패킷(packet)</b>이라고 부른다.
- 패킷은 목적지 종단 시스템으로 네트워크를 통해 보내지고 목적지에서 원래의 데이터로 다시 조립된다.

<br>
<br>

### 스위치 (switch, 패킷 교환기)
패킷 교환기(=스위치)는 입력 통신 링크 중 하나로 도착하는 패킷을 받아서 출력 통신 링크 중 하나로 그 패킷을 전달한다. \
대표적으로 라우터(router)와 링크 계층 스위치(link-layer switch)의 두 종류가 가장 널리 사용된다.
- 두 종류 모두 최종 목적지 방향으로 패킷을 전달

<br>

- 링크 계층 스위치는 보통 **접속 네트워크**에서 사용
   > 접속 네트워크(access network)는 한 종단 시스템에서 다른 종단 시스템까지의 경로상에 있는 첫 번째 라우터를 말한다.
   - 특정 네트워크 내에서 **Mac address**를 사용하여 올바른 디바이스로 data frame을 전송한다.
- 라우터는 **네트워크 코어**에서 사용
   > 네트워크 코어(network core)는 전체 네트워크 시스템에 중앙에 위치하여 데이터를 전송하는 핵심적인 역할을 수행하고, \
   > 수많은 라우터들이 그물처럼 얽혀있는 구조로, 패킷을 올바른 곳으로 전송한다.

<br>

> 패킷이 송신 종단 시스템에서 수신 종단 시스템에 도달하는 동안 거쳐온 일련의 통신 링크와 패킷 스위치를 \
> 네트워크 상의 <b>경로(route 혹은 path)</b>라고 한다.

<br>
<br>

### ISP (Internet Service Provider)
종단 시스템은 ISP를 통해 인터넷에 접속한다.
- ISP는 패킷 스위치와 통신 링크로 이루어진 네트워크
- 종단 시스템에게 다양한 네트워크 접속을 제공 (케이블 모뎀, 가정용 초고속 접속(DSL), 고속 LAN 접속, 이동 무선 접속 등)
- CP(content provider)에게 인터넷 접속을 제공 -> 웹사이트, 비디오 서버를 인터넷에 직접 연결할 수 있게 된다.

<br>

인터넷은 종단 시스템을 서로 연결하는 것이므로, 종단 시스템에 인터넷 접속을 제공하는 ISP들도 서로 연결되어야 한다.
- 하위 계층 ISP는 국가 및 국제 상위 계층 ISP를 통해 서로 연결한다.
- 상위 계층 ISP들은 서로 직접 연결된다.
   - 상위 계층 ISP는 **광 링크로 연결된 고속 라우터**로 구성된다.
- 상위 · 하위 계층과 무관하게 각 ISP 네트워크는 **IP 프로토콜 수행하고 네이밍(naming)과 주소배정 방식을 따른다.**


<br>
<br>

### 프로토콜 (protocol)
프로토콜은 인터넷에서 정보 송수신을 제어한다. \
종단 시스템, 패킷 스위치를 비롯한 인터넷의 여러 구성요소는 여러 프토로콜을 수행한다.
- TCP와 IP는 인터넷에서 가장 중요한 프로토콜
   - TCP(Transmission Control Protocol)
   - IP(Internet Protocol)
      - 라우터와 종단 시스템 사이에서 송수신되는 패킷 포맷을 기술한다.
> 두 주요 프로토콜을 통칭하여 **TCP/IP**라고 한다.
