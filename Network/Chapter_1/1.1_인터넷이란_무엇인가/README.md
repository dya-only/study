- 분산 애플리케이션에 서비스를 제공하는 네트워크 인프라 관점에서 설명하기

## 1.1.1 구성요소로 본 인터넷

인터넷은 전 세계적으로 수십억 개의 컴퓨팅 장치를 연결하는 컴퓨터 네트워크이다.
데스크톱 PC, 리눅스 워크스테이션, 서버 등의 전통적인 컴퓨팅 장치와, 스마트폰, 태블릿, 가전제품 등의 비전통적인 인터넷 **사물들**이 모두 인터넷에 연결되고 있다.

> 이러한 모든 장치를 인터넷 용어로 **호스트(host)** 혹은 **종단 시스템(end system)** 이라고 한다.

종단 시스템은 **통신 링크(communication link)**와 **패킷 스위치(packet switch)**의 네트워크로 연결된다.
- 이러한 링크는 동축 케이블, 구리선, 광케이블, 라이도 스펙트럼을 포함한 다양한 물리 매체로 구성된다.
